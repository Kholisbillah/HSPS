## I. PETUNJUK

1. Periksalah dengan teliti dokumen soal ujian praktik.

   PT. Cakrawala Global Yaksa telah menerima order pembuatan dan pengembangan aplikasi berbasis web dari berbagai konsumen. Anda sebagai seorang **Full Stack Web Programming** diminta oleh seorang *Project Manager* untuk membuat dokumen analisis serta aplikasi berbasis client – server untuk aplikasi berbasis web sesuai dengan case project yang anda pilih.

   Ujian praktik ini terdiri rincian penugasan sebagai berikut:
2. 1. Melakukan analisis kebutuhan untuk requirement aplikasi yang akan dibuat atau dikembangkan
   2. Membuat Pemodelan Sistem Berorientasi Obyek (SBO) dengan UML (*Unified Modelling Language*) dan Pemodelan Database Object (PDBO) dengan database MySQL/Oracle/Ms Access
   3. Membuat rancangan interface Input – Process – Output (Mock Up)
   4. Membuat perintah dasar basis data SQL (DDL, DML dan DCL)
   5. Membuat basis data SQL – MySQL
   6. Membuat basis data transaksi (*Trigger, Function, Procedure, View*)
   7. Melakukan export dan import basis data
   8. Melakukan pengkodean dalam bahasa pemrograman PHP (Native dan atau Framework: Laravel, CodeIgniter 3, CodeIgniter 4, dll)
   9. Menerapkan CSS (*Cascade Style Sheet Internal & External*)
   10. Menerapkan Bootstrap untuk pemrograman web
   11. Melakukan koneksi basis data (MySQL dan MySQLi)
   12. Menerapkan Library PHP (Chart, Resize File, Rename, Shop Cart, Tracking, PDF)
   13. Menerapkan JavaScript dan Librarinya (JQuery)
   14. Menerapkan Export dan Import data (PDF, Excel) dalam pemrograman web
   15. Menerapkan JSON dan API dalam pemrograman web
   16. Melakukan registrasi domain dan WHM (*Web Hosting Management*)
   17. Menerapkan pemrograman web dalam GitHub
   18. Membuat dokumentasi berupa *User Guide Application*
3. Hak akses user untuk aplikasi terdiri dari **Super Root**, **Administrator** dan **User**
4. Peralatan utama komputer telah disediakan oleh masing-masing.
5. *File* data pendukung dicari dibuku dan internet.
6. Program aplikasi yang bersifat khusus disiapkan oleh peserta.
7. *File* teks, video, special efek, animasi dan gambar disesuaikan dengan kebutuhan.
